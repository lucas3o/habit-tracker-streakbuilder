# StreakBuilder — Premium Habit Tracker

StreakBuilder is a beautiful, dark-mode Habit Tracker web application built using the MERN stack (**Node.js, Express, MongoDB, and React/Vite**). It features client-side timezone streak calculations, categories, theme colors, and an interactive 7-day weekly log.

## Features
* 🔄 **Custom Categories**: Organize habits into Routine, Fitness, Mind, Work, or Health.
* 🎨 **Color Themes**: Customize each habit card with Emerald, Indigo, Rose, Amber, or Violet theme colors.
* 📅 **Interactive 7-Day Log**: Toggle habit completions for today or any of the past 7 days directly on the card.
* 📈 **90-Day Activity Heatmap**: Visualize progress over the last 3 months.
* 🔥 **Streak Calculations**: Accurate tracking of current and longest streaks.
* 🔒 **User Authentication**: JWT-based secure sign up and sign in.

## Technologies Used
* **Frontend**: React, Vite, React Router, Lucide Icons, CSS Custom Properties
* **Backend**: Node.js, Express.js, MongoDB (Mongoose), JWT, BcryptJS
* **Database**: MongoDB (Local Instance)
